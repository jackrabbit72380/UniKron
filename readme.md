# 🌌 UniKron Dashboard 

Built For the Future!

### A Unified, Cross‑Platform Interface Experience

UniKron is the modern evolution of a long‑running vision: a cinematic, customizable, controller‑first interface for browsing and launching games across PC, console‑style devices, and handheld systems. Built as a theme for the Pegasus Frontend, UniKron delivers a seamless, art‑driven experience powered by modern UI technologies.

---

## 🌐 Project Uniflow: The Origin

Project Uniflow began as a revolutionary collection of themes designed to unify the look and feel of PC and console interfaces. Its mission was simple:  
**create a cohesive visual experience across all devices and applications.**

Uniflow themes were built to be:

- Visually consistent  
- Highly customizable  
- Compatible across multiple platforms  
- Intuitive for both controller and touchscreen navigation  

From minimalistic layouts to bold, expressive designs, Uniflow empowered users to personalize their digital environments with ease.

---

## 🏁 Early History: From CEMUgui to UniFlow

The project began under the name **CEMUgui**, inspired by several influential frontends:

- CemUI  
- MiiFlow  
- WiiFlow  
- The CoverFlow Project  
- Freestyle Dashboard  
- Aurora Dashboard  

These early inspirations shaped the foundation of UniFlow before it grew into its own identity.

---

## 🖥️ UniFlow (AutoIt Era)

UniFlow was originally a standalone Windows application built using:

- **AutoIt** (a basic scripting language)  
- **IIRCHET 3D Engine**  

Its features included:

- Coverflow‑style layouts  
- Controller + touchscreen navigation  
- Automatic scraping of box art and metadata  
- Emulator launching  
- Customizable UI elements  

Despite its ambition, AutoIt imposed major limitations:

- Slow development  
- Limited debugging tools  
- No hardware acceleration  
- Windows‑only compatibility  
- Difficulty scaling complex UI systems  

Still, UniFlow persisted for years, driven by a clear vision of a unified, cinematic frontend.

---

## 🚀 The Breakthrough: Pegasus Frontend + QML

The project reached its true potential with **Pegasus**, an open‑source, cross‑platform frontend that runs on:

- Windows  
- Linux  
- macOS  
- Android  
- Raspberry Pi  
- Odroid  
- Other embedded devices  

Pegasus provides:

- Full hardware acceleration  
- SDL2 or Qt Gamepad input  
- High‑performance rendering  
- Cross‑platform consistency  
- Modern UI architecture  

This opened the door to a complete transformation of the project.

---

## 🌌 UniKron: The Modern Evolution

UniKron is built using:

- **QML + QtQuick2**  
- **Qt 5.15+**  
- **C++11‑compatible compiler**  
- **Multimedia, SVG, SQL (SQLite v3)**  
- **SDL2 or Qt Gamepad**  

This modern stack enables:

- Declarative animations  
- Reactive layouts  
- Clean scaling across resolutions  
- Unified input handling  
- Smooth performance on all devices  

### UniKron is not a new project—  
**it is the true form of what UniFlow always wanted to be.**
